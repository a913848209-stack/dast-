# DAST-Scanner

一款轻量级、自动化、企业级的 Web 动态应用安全扫描（DAST）工具。

本工具旨在通过非破坏性方式，帮助开发者、安全工程师在等保 2.0 合规、上线前巡检及 CI/CD 流水线中快速发现常见的 Web 安全风险。

## 🚀 核心能力
- **合规审计**：自动化检测安全头 (HSTS, CSP 等)、Cookie 属性、传输协议及 HTTPS 配置。
- **漏洞检测**：启发式探测 SQL/NoSQL 注入、XSS、SSRF、CRLF 注入、开放重定向及路径遍历。
- **配置扫描**：探测敏感路径 (如 .env, .git, actuator)、目录列举及 GraphQL 风险。
- **CI/CD 友好**：支持安静模式与退出码判断，轻松接入自动化流水线。
- **轻量易用**：基于 Python 标准库，无强制依赖，支持自定义认证头，规避 Token 泄露风险。
